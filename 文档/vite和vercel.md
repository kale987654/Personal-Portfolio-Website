### ⚡ Vite：下一代前端构建工具

**它是什么？**
你可以把 Vite 想象成一个**超级智能的、速度极快的开发助手**。你给它你的源代码（HTML, CSS, JS），它会帮你：
1.  **启动一个本地开发服务器**，让你在浏览器里预览网站。
2.  **提供极速的热更新**：你修改代码后，浏览器页面**瞬间更新**，无需手动刷新。
3.  **在打包时**，帮你压缩、优化代码，让它变得又小又快。

**为什么它比传统工具（如 Webpack）好？**
-   **快！快！快！** 启动速度和热更新速度是数量级的提升。
-   **配置简单**：几乎零配置就能开始一个项目。

**如何使用它？（只需3步）**
1.  **安装**：你需要先安装 Node.js（一个JavaScript运行环境），然后通过它的包管理器 npm 来安装 Vite。
    ```bash
    # 在你的项目目录下打开终端，执行：
    npm create vite@latest
    ```
2.  **回答几个问题**：它会问你项目名、框架（选 Vanilla，即原生JS）、语言（选JavaScript）。
3.  **运行**：按照提示运行几条命令。
    ```bash
    cd your-project-name # 进入项目文件夹
    npm install          # 安装依赖
    npm run dev          # 启动开发服务器
    ```
4.  浏览器会自动打开 `http://localhost:5173`，你的项目就在那里了！现在你可以开始编码，任何保存都会实时显示在浏览器中。

### 🚀 Vercel：极简的部署平台

**它是什么？**
Vercel 是一个**为你提供网站托管和部署的服务**。你把它和你 GitHub 上的代码仓库连接起来，之后：
-   每次你写完代码，**只需推送到 GitHub**。
-   Vercel 会自动检测到变化，**自动开始打包、部署**。
-   几十秒后，你的网站就有了一个**免费的、能公开访问的线上地址**（如 `my-portfolio.vercel.app`）。

**它有什么好处？**
-   **完全免费**：对于个人项目完全够用。
-   **无限简单**：无需自己买服务器、配置Nginx等复杂操作。
-   **自动HTTPS**：给你的网站自动配上安全证书（那个小锁图标）。
-   **全球加速**：你的网站会被分发到全球的服务器，任何人访问都很快。

**如何使用它？（只需4步）**
1.  将你的代码上传到 GitHub（创建一个新的仓库并推送上去）。
2.  访问 [vercel.com](https://vercel.com/)，用 GitHub 账号注册。
3.  点击 **“Add New...” -> “Project”**。
4.  导入（Import）你的 GitHub 仓库，所有配置都用默认的，直接点击 **“Deploy”**。

**就这样！** 等待几分钟，你的个人作品集就向全世界展示了。

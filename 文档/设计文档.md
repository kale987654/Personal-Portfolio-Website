### 📁 网站核心页面与模块

你的作品集可以设计成**单页应用（Single Page Application）**，通过导航跳转到不同章节。主要模块如下：

1.  **导航栏 (Navbar)**：固定顶部，包含Logo、导航链接（首页、项目、关于、联系）、暗黑模式切换按钮。
2.  **英雄区 (Hero Section)**：大图背景，你的姓名、职位、一句酷炫的标语、以及“下载简历”按钮。
3.  **项目展示区 (Projects Section)**：用Grid或Flexbox展示3-6个你的最佳项目（每个项目有图、标题、描述、技术栈、项目链接）。
4.  **技能区 (Skills Section)**：用图标或进度条的形式展示你的技术栈（HTML, CSS, JS, Vue等）。
5.  **关于/联系区 (About/Contact)**：简短自我介绍、联系方式（邮箱、GitHub链接、社交媒体）。
6.  **页脚 (Footer)**：版权信息、再次放置社交链接。

### 🚀 功能实现要点

#### 1. 响应式布局 + 移动端适配
-   **策略**：**移动优先（Mobile First）**。先写移动端样式，再用媒体查询（`@media`）适配平板和桌面。
-   **断点参考**：
    ```css
    /* 手机 (默认) */
    /* 平板 */
    @media (min-width: 768px) { ... }
    /* 笔记本电脑 */
    @media (min-width: 1024px) { ... }
    /* 台式机 */
    @media (min-width: 1280px) { ... }
    ```
-   **具体变化**：
    -   导航栏在移动端变为汉堡菜单。
    -   项目展示从单列变为多列。
    -   字体大小和内边距随屏幕增大而调整。

#### 2. 暗黑模式
-   **核心逻辑**：通过JS切换HTML标签上的类名（如 `dark-mode`），CSS变量定义两套主题。
-   **代码结构**：
    ```css
    :root {
      --bg-color: #fff;
      --text-color: #333;
      /* 定义一套浅色主题变量 */
    }
    .dark-mode {
      --bg-color: #1a1a1a;
      --text-color: #fff;
      /* 定义一套深色主题变量 */
    }
    body {
      background-color: var(--bg-color);
      color: var(--text-color);
      transition: all 0.3s ease; /* 添加过渡动画 */
    }
    ```
    ```js
    const toggleButton = document.getElementById('theme-toggle');
    toggleButton.addEventListener('click', () => {
      document.body.classList.toggle('dark-mode');
    });
    ```

#### 3. 简历下载
-   **实现**：将你的PDF简历文件放在项目文件夹中（如 `/assets/resume.pdf`）。
-   **HTML**：使用 `<a>` 标签，并添加 `download` 属性。
    ```html
    <a href="/assets/resume.pdf" download class="download-btn">下载简历</a>
    ```
-   **注意**：在Vercel/Netlify上部署时，确保文件路径正确。

#### 4. 平滑滚动与导航
-   **实现**：为每个章节（如 `#projects`）设置id，导航链接指向这些id（如 `#projects`）。
-   **CSS**：可添加CSS实现平滑滚动。
    ```css
    html {
      scroll-behavior: smooth;
    }
    ```

### 📅 开发流程建议

1.  **第1步：设计与规划 (1天)**
    -   在Figma或纸上画出线框图（Wireframe），规划好不同屏幕尺寸下的布局。
    -   确定你的配色方案（浅色 & 深色）。

2.  **第2步：搭建静态结构 (2天)**
    -   用HTML搭建所有页面的静态结构。
    -   用CSS（Grid+Flexbox）实现桌面端的布局。**先不要做样式细节**。

3.  **第3步：实现响应式 (2天)**
    -   添加媒体查询，完成手机和平板的布局适配。

4.  **第4步：添加视觉样式与功能 (2天)**
    -   完善细节样式（颜色、阴影、圆角等）。
    -   用JavaScript实现暗黑模式切换。
    -   完成简历下载、导航跳转等功能。

5.  **第5步：测试与部署 (1天)**
    -   在不同的设备和浏览器上测试。
    -   使用Vite构建，并部署到Vercel/Netlify。

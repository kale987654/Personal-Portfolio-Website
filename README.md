# 个人作品集网站

一个现代化、响应式的个人作品集网站，使用原生HTML5、CSS3和JavaScript开发。

## 功能特性

- 多主题支持（深蓝、森林绿、石板灰）
- 完全响应式设计
- 原生JavaScript，无框架依赖
- 模块化代码结构
- 简历下载功能
- 平滑滚动和动画效果
- 易于自定义和扩展

## 技术栈

- **前端**: HTML5, CSS3, JavaScript ES6+
- **工具**: Git, Webpack(准备), Node.js(基础)
- **设计**: Figma, Photoshop
- **文档**: Markdown
